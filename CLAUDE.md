## Quality Standard

Code quality matters. This module aims to be the best database query layer available — reliable, secure, ergonomic, and well-documented. We want this to be a module the community wants to contribute to.

## What This Project Is

DBIx::Wizard is a thin, fluent database query executor for Perl that sits between raw DBI and DBIx::Class. It uses SQL::Wizard (at `/home/thomas/work/perl-SQL-Wizard/`) as its SQL generation backend.
